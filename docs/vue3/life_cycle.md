# 谈谈你对vue生命周期的理解

1. 在Vue中，生命周期是指从组件创建到销毁的整个过程。在这个过程中，Vue会触发一系列的钩子函数，我们可以在这些函数中插入自己的代码，以实现各种想要的功能。
2. 如果是选项式API，生命周期钩子函数分别为beforeCreate，created，beforeMount，mounted，beforeUpdate，updated，beforeUnmount和unmounted函数
